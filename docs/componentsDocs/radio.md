## Radio  单选框

<ClientOnly>
  <dz-radio/>
<font size=5>Attributes</font>
| 参数| 说明 | 类型 | 可选值 | 默认值 |
| :------ | ------ | ------ | ------ | ------ |
| value / v-model | 绑定值 | string / number / boolean | - | - |
| label | Radio 的 value | string / number |- | - |
| disabled | 是否禁用 | boolean |- | false |
| size | 多选框组尺寸，仅对按钮形式的 Checkbox 或带有边框的 Checkbox 有效 | string | medium / small / mini | - |

</ClientOnly>
